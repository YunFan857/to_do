# 📝 我的待办

> 一个简洁优雅的**本地待办清单**，单文件 HTML 应用，专为个人日常使用设计。

## ✨ 特性

- **四类待办，按周期管理**
  - 📌 **一次性** — 有截止日期的任务，自动标记过期
  - ☀️ **每日** — 每天循环，自动统计「连续打卡」
  - 📅 **每周** — 按指定星期循环
  - 🗓️ **每月** — 按指定日期循环
- **圆角小卡片** — 清新现代的视觉设计
- **本地存储** — 数据存浏览器 `localStorage`，**零依赖、零联网**
- **数据备份** — 一键导出 / 导入 JSON 备份
- **键盘快捷键** — `N` 新建、`Esc` 关闭

## 🚀 使用方式

### 桌面图标（macOS，已生成）

直接双击桌面上 **「我的待办」** 应用图标即可启动。

### 直接打开

双击 `index.html`，浏览器自动打开。

### 本地开发

```bash
cd path/to/to_do
python3 -m http.server 8000
# 浏览器访问 http://localhost:8000
```

## ⌨️ 快捷键

| 按键 | 功能 |
| --- | --- |
| `N` | 新建待办（焦点不在输入框时） |
| `Esc` | 关闭弹窗 |
| `Space` / `Enter` | 切换完成状态（卡片聚焦时） |

## 🛠 技术栈

- 纯 HTML / CSS / JavaScript
- **零依赖** — 无 npm、无打包工具
- `localStorage` 持久化
- 响应式布局，桌面 / 平板 / 手机皆可

## 📦 数据存储

所有待办存于浏览器 `localStorage`，键名 `todo-app-v1`。

> ⚠️ 清浏览器数据 = 清空所有待办。**请定期用「导出」按钮备份**。

## 🔄 备份与恢复

- **导出**：顶部「导出」→ 下载 `todo-backup-YYYY-MM-DD.json`
- **导入**：顶部「导入」→ 选择备份文件 → 确认覆盖
