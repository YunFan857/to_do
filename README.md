# 📝 我的待办

> 一个简洁优雅的**本地待办清单**，单文件 HTML 应用，专为个人日常使用设计。

## ✨ 特性

- **五类待办，按周期管理**
  - 📌 **一次性** — 有截止日期的任务，自动标记过期
  - ☀️ **每日** — 每天循环，自动统计「连续打卡」
  - 📅 **每周** — 按指定星期循环
  - 🗓️ **每月** — 按指定日期循环
  - 🎂 **每年** — 按指定月日循环
- **圆角小卡片** — 清新现代的视觉设计
- **🌏 首都时间** — 可切换中国 / 美国，实时显示北京或华盛顿特区时间，并自动适配夏令时
- **本地存储** — 数据保存到 macOS 本地应用数据文件，不依赖浏览器缓存
- **数据备份** — 一键导出 / 导入 JSON 备份
- **🔔 定时提醒** — 提醒跟随当前选择的首都时区，到点后播放声音 + 弹出系统通知 + 顶部横幅
- **键盘快捷键** — `N` 新建、`Esc` 关闭

## 🚀 使用方式

### 桌面图标（macOS，已生成）

直接双击桌面上 **「我的待办」** 应用图标即可启动。

### 直接打开

双击 `index.html`，浏览器自动打开。

### 本地开发

```bash
cd path/to/to_do
python3 -m http.server 8000
# 浏览器访问 http://localhost:8000
```

## ⌨️ 快捷键

| 按键 | 功能 |
| --- | --- |
| `N` | 新建待办（焦点不在输入框时） |
| `Esc` | 关闭弹窗 |
| `Space` / `Enter` | 切换完成状态（卡片聚焦时） |

## 🔔 提醒功能

鼠标悬停任意卡片，点击右上角 **🔕** 按钮即可设置提醒。

- 一次性：到 `截止日期` 的指定时间提醒一次
- 每日：每天到点提醒，完成当日打卡后当天不再响
- 每周：每周指定星期几到点提醒
- 每月：每月指定日到点提醒
- 每年：每年指定月日到点提醒

到点时会同时触发：
- 播放一段柔和的提示音（Web Audio，无需音频文件）
- 弹出系统通知（需授权浏览器通知权限）
- 页面顶部滑下横幅提示（始终可见，不受权限影响）

提醒时间跟随页面顶部选择的时区：选择中国时，`10:00` 代表北京时间上午 10 点；选择美国时，`10:00` 代表华盛顿特区时间上午 10 点。切换时区后，已有提醒会立即按新时区重新排程；华盛顿特区的夏令时会自动处理。

首次设置提醒时会请求通知权限，**允许后系统通知才会在后台弹出**。
> 浏览器 / 系统通知在 Safari 中对 `file://` 站点支持有限，建议使用 Chrome / Edge / Brave 等以获得完整体验。

## 🛠 技术栈

- 纯 HTML / CSS / JavaScript
- **零依赖** — 无 npm、无打包工具
- macOS 本地 JSON 文件持久化
- 响应式布局，桌面 / 平板 / 手机皆可

## 📦 数据存储

桌面应用会启动一个仅监听本机的本地服务，并打开本地页面；所有待办保存到：

```text
~/Library/Application Support/我的待办/data.json
```

因此清理 Chrome / Safari 的浏览器数据不会清空待办。首次使用新版桌面应用时，会自动把旧版 `file://` 页面中的待办迁移到这个文件。直接双击 `index.html` 时如果本地服务未启动，会降级使用浏览器临时存储；请通过桌面上的「我的待办」应用图标启动，才能使用本地文件存储。

## 🔄 备份与恢复

- **导出**：顶部「导出」→ 下载 `todo-backup-YYYY-MM-DD.json`
- **导入**：顶部「导入」→ 选择备份文件 → 确认覆盖
